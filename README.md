# Excel VBA + Web API Examples

This repository contains ready-to-use Excel files demonstrating how to call public Web APIs directly from VBA.

Each `.xlsm` file is fully functional — just download, enable macros, and explore.

Each demo shows:
- Simple **WinHTTP** calls in pure VBA  
- Lightweight JSON handling (no external libs)  
- Optional fallback logic or free-tier API key usage  
- Practical examples for Excel users
